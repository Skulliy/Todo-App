You can run the app using:
ng serve 
&
npm run server
The app has the following functions
1)The ability to add tasks via the form and save task button.
2)The ability to remove tasks via the small x button on the right.
3)The ability to set a reminder via the set reminder checkbox during creation or the reminder button.
